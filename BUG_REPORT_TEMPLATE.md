# Bug Report Template

バグが出た時は、この形でメモすると原因追跡しやすいです。

## バージョン
例: v1.2.0

## 端末/環境
例: iPhone / Simple Server / Safari

## 何が起きたか
例: Wave 7で中ボスを倒した後、次のWaveに進まない。

## 期待した動き
例: アーティファクト選択が出て、その後Wave 8に進む。

## 再現手順
1. やわらか森に潜る
2. Wave 7まで進める
3. 中ボスを倒す
4. 画面が止まる

## 直前に選んだスキル/アーティファクト
例: ぽかぽか領域、星ふりパーティ、むらさきエンジン

## debug表示
URL末尾に `#debug` を付けて再現できる場合、debugパネルの内容を記録。
